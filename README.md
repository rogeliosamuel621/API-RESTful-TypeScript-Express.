# SocialMedia-API

## Getting started

### Install dependencies
```
npm install
```

### Development
```
npm run dev 
```

### Production
```
npm run build 
```

``` 
npm run production 
```

## Built with
- Node
- Typescript
- MySQL

## Features
-  Authentication with jwt
- hash passwords with bcryptjs
- validate data schema with express validator
- development language: Typescript
- Middlewares
- API key
